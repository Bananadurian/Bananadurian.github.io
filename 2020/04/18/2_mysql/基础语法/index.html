其它

显示数据库表的建表信息
show create table table_name

查询数据库表的属性
SHOW COLUMNS FROM table_name:

显示数据表的属性，属性类型，主键信息 ，是否为 NULL，默认值等其他信息。
如：show columns from table_name;

数据表的详细索引信息
SHOW INDEX FROM table_name

显示数据表的详细索引信息，包括PRIMARY KEY（主键）。

order by排序
ASC（升序），DESC（降序，新的在前面）

like（注意FIND_IN_SET查询方法）
查询以 java 字段开头的信息。
SELECT * FROM position WHERE name LIKE 'java%';
查询包含 java 字段的信息。
SELECT * FROM position WHERE name LIKE '%java%';
查询以 java 字段结尾的信息。
SELECT * FROM position WHERE name LIKE '%java';

注意sql默认字符需要使用‘ ’括起来

distinct 针对某个字段去重
distinct id
count（distinct id）

limit

注意limit分页的不包含起始位置，从0开始

SELECT column_name(s) FROM table_name LIMIT number;
返回一条数据
select * from table_name limit 1
检索记录行 6-15（取10条数据，从第6条开始拿）
SELECT * FROM table LIMIT 5,10;
检索记录行 96-last
SELECT * FROM table LIMIT 95,-1;

为了检索从某一个偏移量到记录集的结束所有的记录行，可以指定第二个参数为 -1

检索前 5 个记录行
mysql> SELECT * FROM table LIMIT 5;

如果只给定一个参数，它表示返回最大的记录行数目：
换句话说，LIMIT n 等价于 LIMIT 0,n。

比较运算符
between、in、 and、or， =、＞、＜、＞=、＜=、<>或!=(不等于)

和where一起使用，注意between的用法
判断是否为空 =‘ ’

时间格式化
当前普通时间
select now()
普通时间转Unix
select unix_timestamp(20190725115634)
Unix转普通时间
select from_unixtime（135454354）

unix_timestamp(now()) 获取当前Unix时间戳

group by
group by id having id > 10

注意函数的使用，如求和计数等，group by分组，以及语句的书写顺序
sum 和 group by 一起用的时候先执行 group by 再执行 sum
having（一般和group by一起使用，解决where不可以和group不可以一起使用的问题）

条件控制语句

IF(expr1,expr2,expr3)
如果 expr1 是TRUE (expr1 <> 0 and expr1 <> NULL)，则 IF()的返回值为expr2; 否则返回值则为 expr3。IF() 的返回值为数字值或字符串值，具体情况视其所在语境而定

IFNULL(expr1,expr2)

假如expr1 不为 NULL，则 IFNULL() 的返回值为 expr1; 否则其返回值为 expr2。IFNULL()的返回值是数字或是字符串，具体情况取决于其所使用的语境。

case when then end

规则如下：1、>=90：优秀 2、>=80：良好 3、>=60：及格 4、<60：不及格
